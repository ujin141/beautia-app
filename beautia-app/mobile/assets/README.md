# 앱 에셋 디렉토리

이 디렉토리에는 앱에서 사용할 이미지 및 아이콘 파일이 포함되어 있습니다.

## 필요한 파일

### 필수 파일
1. **icon.png** - 앱 아이콘 (1024x1024px)
2. **splash.png** - 스플래시 스크린 이미지
3. **adaptive-icon.png** - Android 적응형 아이콘 (1024x1024px)
4. **favicon.png** - 웹 파비콘

### 권장 파일
- **icon-background.png** - 아이콘 배경
- **icon-foreground.png** - 아이콘 전경

## 파일 규격

### icon.png
- 크기: 1024x1024px
- 형식: PNG (투명 배경 권장)
- 플랫폼: iOS, Android, Web 공통

### splash.png
- 크기: 1242x2436px (iPhone X 기준) 또는 2048x2732px
- 형식: PNG
- 배경: #FFFFFF 또는 브랜드 색상

### adaptive-icon.png (Android)
- 크기: 1024x1024px
- 형식: PNG
- 배경: 단색 권장

## 생성 방법

1. **디자인 툴 사용** (Figma, Adobe XD, Sketch 등)
2. **온라인 생성기 사용**
   - [App Icon Generator](https://www.appicon.co/)
   - [Icon Kitchen](https://icon.kitchen/)
3. **AI 생성기 사용**
   - DALL-E, Midjourney 등으로 아이콘 디자인 생성 후 편집

## 임시 파일

개발 중에는 플레이스홀더 이미지를 사용할 수 있습니다. 하지만 실제 배포 전에는 정식 디자인이 필요합니다.
